# 鉾立口交通指南

## 概要
该登山口位于秋田县仁贺保市，地处鸟海Blue Line公路秋田县一侧的最高点，是使用者最多的登山口，也是象泻路线的起点。

## 自驾车
- **路线**: 从日本海东北自动车道“象泻IC”驶出，沿鸟海Blue Line公路行驶。
- **所需时间**: 从象泻IC约需35分钟。
- **停车场**: 可使用鉾立停车场（约250个车位，免费）。
- **注意事项**: 鸟海Blue Line公路在冬季关闭。夏季周末停车场会非常拥挤。

## 公共交通
- **最近车站**: JR羽越本线“象泻站”。
- **合乘出租车**: 可从JR象泻站乘坐季节性运行的合乘登山出租车。需要预约。
